En la sección de “Movimientos” se tienen los modulos de (que se visualizan en el menú lateral izquierdo): facturación, cotización, anulación, consultas, recibos, ingreso caja, salida caja, egresos, orden pago, cotización pendiente.
 
En la seccion de “Stock” se tienen los modulos de: Pedido Nuevo, Modificar Pedido, Informe Pedidos, Salida Stock, Ingreso Stock, Inventario.
 
Modulo Facturación (de sección Movimientos): Al presionar el botón Facturación se ingresa a este modulo. 
Para poder acceder a este modulo es necesario abrir caja (botón caja). 
Se abre la pestaña “Emision Comprobantes” que es un formulario donde hay que seleccionar:
•	Tipo de consumidor (menú desplegable con los clientes cargados con la informacion: “Denominacion”, “Condicion IVA” y “CUIT”)
•	Vendedor (menú desplegable con nombres de vendedores)
•	Articulo (menú desplegable con Articulos listados, incluyendo código, nombre, precio publico, precio mayorista, stock actual, cod rapido)
•	Cantidad (menú desplegable con un numero)
•	Codigo barra (Solo habilitado si seleccionamos el item “Codigo de barra” en el menú de “Emision” Superior)
 
Al ingresar a este modulo se abre una nueva pestaña del menú superior “EMISION”, que muestra las secciones:
•	Comprobante (Indica tipo de factura y nro)
•	Numero

Tambien están las ventanas de acceso rápido “Pagos”, “Descuentos”, “Promociones”,”Importes”. 
Si se presiona el botón “Pagos”: Se abre la ventana “Formas de pago” con la lista a la izquierda con todas las formas de pago:
•	Efectivo
•	Cta.Cte.$
•	Tarjeta DEBITO
•	Tarjeta CREDITO
•	Cheque
•	Transf. Bancaria
Y debajo se muestra el importe y dos botones “Cancela” y “Confirma”. Al seleccionar cada forma de pago, a la derecha se muestra el formulario para cargar los datos de esa forma de pago. Para tarjeta de crédito muestra un menú desplegable “cuotas”, “lote”, “nro cupon”, y debajo un cartel verde “Recargo financiero” que muestra el valor del recargo para la los datos de tarjeta y cuotas seleccionado, y la cantidad de cuotas con el valor de cada cuota.
Otra forma de cargar productos es seleccionando el Item “Codigo de Barra”. Nos habilita en campo “Codigo de barra” para introducir manualmente el codigo
Una vez rellenado el formulario se puede oprimir el botón “FACTURA” del menú superior “EMISION”. Se muestra el ventana “FORMA DE PAGO” con el texto “Confirma FACTURA en EFECTIVO?” y dos botones: “Si” y “No”. Si se selecciona “Si” vamos la ventana “Descuento, vuelto” que muestra:
•	forma de pago seleccionada
•	importe a pagar 
•	Descuento Pie Factura (para rellenar, predeterminado en 0)
•	Nuevo importe a pagar (después del descuento)
•	Paga con (para introducir el valor con el que paga el cliente)
•	Vuelto (calculado)
•	Dos botones debajo: “Rechazar” y “Aceptar”
Al darle a “Aceptar”: Se abre el siguiente ventana “Pto Venta y Nro Comprobante”, y debido a que la carga es manual se muestran 2 campos para rellenar:
•	Punto de Venta
•	Numero de Comprobante
•	Dos botones debajo: “Rechazar” y “Aceptar”

Al darle a “Aceptar”: Se muestra el ventana “Generado correctamente” y dos botones “Imprime” y “Aceptar”

Modulo Cotización (de sección Movimientos): Al presionar el botón Cotización se ingresa a este modulo. 
El formulario que se muestra es exactamente el mismo que para Facturacion, pero cambia el menú superior, cambiando el botón “FACTURA” por los botones “Cotización” y Factura”, lo explico mas detalladamente:
Al ingresar a este modulo se abre una nueva pestaña del menú superior “EMISION”, que muestra las secciones:
•	Numeración
•	Generar:
o	Boton COTIZACION
o	Boton PRESUPUESTO
•	Acciones
o	Boton Borrar Articulo
•	Utilizar
o	Consumidor final (seleccionador con tick)
o	Codigo de barra (seleccionador con tick)
o	Lista P. Mayorista (seleccionador con tick)
•	Mensajes (vacio por el momento)
•	Boton Salir
 
Una vez cargados los datos y se presiona “COTIZACION” Se muestra el ventana “COTIZACION Generada correctamente” y dos botones “Imprime” y “Aceptar”. Si se le da a aceptar se genera la cotización en la ventana inferior y queda cargada. Si luego se efectua la venta, el usuario puede ir a la ventana de facturación y al presionar el botón del menú superior “Consulta Cotizacion” se muestra una lista de las cotizaciones pendientes cargadas donde puede seleccionar una cotización y se rellena automáticamente el formulario de facturación con esos datos y continuar con el proceso normal de facturación, debajo de la lista de cotizaciones están los 3 botones: “ESC”, “Consultar nuevas cotizaciones” (básicamente recarga a ver si se cargaron nuevas cotizaciones pendientes) y “Confirmar”. 






Modulo Pedido Nuevo (de sección Stock): Al presionar el modulo “Pedido Nuevo” se muestra un formulario que incluye
•	Articulos (lista desplegable de artículos)
•	Cantidad (input para números enteros)
•	Boton “Confirma Articulo” (al presionarlo se añade el articulo a la ventana inferior)
•	Estado del pedido
Al ingresar a este modulo se abre una nueva pestaña del menú superior “PEDIDO”, que muestra las secciones:
•	Menu (Solo muestra el texto “Nuevo pedido”)
•	Grilla
o	Boton Borrar Articulo
•	Comprobante (Muestra el nro de pedido)
•	Acciones
o	Boton Generar Pedido
o	Boton Imprime
•	Obtener
o	Boton Stock COMPLETO
o	Boton Stock MINIMO
•	Punto de Reposición
o	Venta de Ultimo Dias (input para ingresar el nro de la cantidad de “últimos días” requerido”, predefinido en 7)
o	Debajo de Stock Minimo  (seleccionador con tick)
o	Boton Obtener Articulos Vendidos
•	Boton Salir
Al ingresar a este modulo se abre el menú inferior “Detalle” en donde podemos visualizar los datos del pedido que se cargan al presionar el botón “Confirma Articulo”. Las columnas son: Codigo, Articulo, Cantidad, Precio Unitario, Total, Stock Actual, Stock Minimo, Stock Pedido, Rubro, Multiplo, Inventario (Indica BAJO, MEDIO o ALTO en colores rojo, amarillo y verde), %CMV.
